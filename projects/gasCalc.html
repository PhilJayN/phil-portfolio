Lessons Learned

  • Take time to give variables proper and descriptive names. This makes reading code more joyful.
  • Learn modular code before tackling a long project. It will make refactoring easier.
  • Draw flow control on a piece of paper so I can refer back to it when debugging. Don't try to keep flow control all in my head.
  • Don't get caught up with regular expression. Sometimes they can help, other times make the situations worse.

Final Note

This app was a challenge to implement because there are many elements that affected each other. When I changed one part of the code, some of the other parts started changing. This is what pushed me to learn modular JavaScript.
