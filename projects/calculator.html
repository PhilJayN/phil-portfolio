Lessons Learned

  • Use less jQuery selectors. DOM is my friend.
  • Comment abstract parts of code. It will be easier to read weeks later.
  • console.log is a guardian angel. Use it often for debugging!

Final Note

My mistake during the early learning process was not starting projects sooner. My learning routine is more straightforward now:

  1. Find programming books with good reviews.
  2. Learn the main concepts, and write down some questions.
  3. Scratch head and lose some hair.
  4. Start on a project. Write down more questions.
  5. Search on Stack Overflow or Reddit if still stuck.
  6. Attend Meetup.com groups and ask others for help.
  7. Profit and celebrate!
